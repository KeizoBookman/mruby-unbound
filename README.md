mruby-unbound
====


## Descriotion
libunbound mruby client.
unbound is DNS cache Server

## sample

c = Unbound.new
c.resolv "forum.mruby.org"
#=> "176.34.60.232"

## Requirement

your env is installed unbound / libunbound

## Install

on your build_config.rb

gem.config :github "KeizoBookman/mruby-unbound"
